What can I upload here?

Option A — a referral file (.json). The structured referral your intake system produces. Same shape as the files in the samples/ folder. Fields it reads:

{
  "patient":   { "name": "...", "dob": "YYYY-MM-DD" },
  "diagnosis_code": "ICD-10 (e.g. M54.50)",
  "requested_cpt":  "CPT/HCPCS (e.g. 72148)",
  "procedure":      "MRI lumbar spine w/o contrast",
  "insurance": { "payer": "...", "member_id": "AET-7781-2200" },
  "npi": "referring provider NPI",
  "raw": "the free-text referral / clinical note"
}

Option B — a clinical note (.txt). Paste/drag the raw referral text; it becomes the note and any fields it can't find get flagged as missing.
